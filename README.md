# Camp Seeker

An example of a CRUD application that allows a user to register an account and search, edit, and add campgrounds to a database.

## Getting Started

### Technologies Used

- Javascript
- EJS
- Cloudinary
- Bootstrap
- Mapbox
- Mongoose
- MongoAtlas
- Flash
- Passport

### Installing

- fork and clone this repo to your local device
- npm install

### Executing program

```text
node app.js
```
